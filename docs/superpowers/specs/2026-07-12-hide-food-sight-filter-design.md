# Скрытие категории «еда» из фильтра достопримечательностей

## Цель

Убрать вариант «еда» только из фильтра подкатегорий во вкладке
«Достопримечательности», поскольку для ресторанов есть отдельная вкладка.

## Изменения

- При построении вариантов фильтра исключить значение `еда` из массива `subs`.
- Оставить категорию `еда` в форме добавления достопримечательности.
- Не менять отображение существующих записей категории `еда` и сохранённые
  данные.

## Проверка

Убедиться, что фильтр не предлагает «еда», остальные категории доступны, а
production-сборка проходит на Node.js 22.
