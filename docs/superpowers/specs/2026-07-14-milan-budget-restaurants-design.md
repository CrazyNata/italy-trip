# Недорогие рестораны Милана

## Цель

Добавить десять не повторяющих текущий список ресторанов Милана для поездки в октябре 2026 года.

## Подборка

- Районы: Ортика, Порта-Венеция, Брера/центр и Навильи.
- Уровень цен: `€` или `€€`.
- Типы мест: траттории, пиццерии и аперитиво.
- Для каждой записи: название, район, Google Maps, координаты, ценовая категория и краткая практичная заметка.
- Не добавлять фотографии: их можно загрузить позднее из редактора ресторана.

## Данные и проверка

Новые записи добавляются в `trip_state.main.payload.data.restaurants` со статусом `хочу`. Перед публикацией проверяются существующие названия и районы, чтобы избежать дублей.
