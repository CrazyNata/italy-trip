# Единое место для фото ресторанов

## Цель

Сделать карточки ресторанов одинаковой высоты и дать каждому месту заметную
область для добавления фотографии.

## Оформление

- Каждая карточка содержит верхнюю область фотографии высотой `220px`.
- Для карточки с фото сохраняется текущий просмотр изображения, фон и действия.
- Для карточки без фото отображается тёплый фон `var(--track)`, иконка камеры и
  подпись «Добавить фото».
- Нажатие на заглушку открывает существующий выбор одного или нескольких
  изображений для этого ресторана.
- Нижняя иконка камеры остаётся, чтобы добавлять изображения и к карточкам, где
  фото уже есть.

## Проверка

После сборки вручную проверить: карточки с фото и без имеют одинаковую верхнюю
высоту, заглушка открывает выбор файла, а загрузка фотографии заменяет заглушку
на существующий фотопросмотр.
