# Исправление пеших маршрутов и фотографий

## Причина

При добавлении Кастель-Гандольфо он оказался первым городом по алфавиту. Экран
пешего маршрута выбирает первый город по умолчанию и показывает римские названия
дней для любого города. Новые места также были добавлены без поля `photo`.

## Изменения

- По умолчанию выбирать город с наибольшим числом мест с назначенным `walkDay`;
  для текущих данных это Рим.
- Для Рима сохранять названия дней «античный Рим», «центр Рима», «Ватикан и
  Боргезе».
- Для других городов показывать нейтральный заголовок «Маршрут по городу».
- Назначить Кастель-Гандольфо и озеро Альбано на первый день маршрута их города
  в порядке: город, затем озеро.
- Записать найденные изображения Wikimedia в поля `photo` обоих новых мест.

## Проверка

После сборки вы вручную проверяете в браузере: по умолчанию открыт Рим с тремя
маршрутами; Кастель-Гандольфо открывается отдельно и показывает два места с
нейтральным заголовком и фотографиями.
