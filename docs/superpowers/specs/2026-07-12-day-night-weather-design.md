# Дневная и ночная погода

## Цель

Показывать в карточке каждого города температуру днём и ночью вместо одного
значения, в обоих режимах погодного блока.

## Данные

- Режим «Сейчас» запрашивает дневной прогноз на текущую дату: максимальную и
  минимальную температуру, а также код погоды.
- Режим «В поездке» запрашивает те же дневные поля для даты посещения; для
  дальних дат используются типичные значения за прошлый год в существующем
  окне дат.
- Температура днём — `temperature_2m_max`, ночью — `temperature_2m_min`.

## Оформление

- Вместо одной крупной температуры карточка показывает строку
  `Днём +24° · Ночью +14°`.
- Иконка и текстовое описание погоды остаются.
- Высота, изображение города, сетка и переключатель режимов не меняются.

## Проверка

После production-сборки пользователь вручную проверяет в браузере оба режима:
у каждого города отображаются дневная и ночная температуры, а при ошибке API
карточка показывает прочерк без ошибки разметки.
