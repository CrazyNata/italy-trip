## Обновление данных

Обновление выполняется атомарно в строке `public.trip_state` с `id='main'`.
Каждая запись добавляется только при отсутствии записи с таким же именем без
учёта регистра. Существующий парк Вилла Боргезе повторно не добавляется. Другие
данные поездки и версия payload не меняются.

## Проверка

После обновления прочитать массивы `restaurants` и `sights`, подтвердить наличие
ровно одной записи каждого нового места и отсутствие дубля Виллы Боргезе.
