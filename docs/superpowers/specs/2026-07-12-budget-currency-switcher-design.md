# Переключатель валют бюджета

## Цель

Показывать бюджет в евро, рублях или чешских кронах и автоматически
пересчитывать суммы при переключении валюты.

## Данные и курсы

- Расходы продолжают храниться в евро без изменения схемы данных.
- При открытии вкладки «Бюджет» приложение загружает актуальные курсы EUR/RUB и
  EUR/CZK из публичного API ExchangeRate-API.
- Загруженные курсы используются только для пересчёта интерфейса и не
  отображаются пользователю.
- При ошибке загрузки доступен только EUR, чтобы не показывать неверные суммы.

## Оформление и поведение

- В шапке бюджета расположены три кнопки `EUR`, `RUB`, `CZK`.
- Выбранная валюта сохраняется в `localStorage` и восстанавливается при
  следующем открытии.
- Переключение меняет общую сумму, «на семью», «в день», суммы строк и символ в
  поле добавления расхода.
- Новый расход вводится в текущей выбранной валюте и перед сохранением
  конвертируется в евро.
- Пользователь не видит курс конвертации.

## Проверка

После production-сборки пользователь вручную проверяет: переключение меняет
все отображаемые суммы, новый расход в RUB/CZK сохраняется и корректно выглядит
после перехода в EUR, а при неудачной загрузке курса доступен EUR.
