# Фильтр категорий на карте ресторанов

## Цель

Дать возможность отбирать рестораны в блоке «Карта ресторанов» по одному или
нескольким форматам: пиццерия, морепродукты, желатерия, бар, ресторан и кафе.
Фильтр действует только внутри блока карты и не меняет основной каталог.

## Данные

В `Restaurant` добавляется необязательное поле `categories?: string[]`.
Допустимы только следующие значения:

- `пиццерия`
- `морепродукты`
- `желатерия`
- `бар`
- `ресторан`
- `кафе`

Отсутствующее поле эквивалентно пустому массиву. Существующие записи не
мигрируются и при включённом фильтре категорий не показываются.

## Редактор ресторана

В модальном редакторе появляется отдельная группа кнопок «Категории». Каждая
кнопка переключает соответствующий тег. Можно выбрать любое количество тегов,
включая ни одного. Тип места (`placeType`) остаётся самостоятельным полем.

## Карта ресторанов

Рядом с выбором города добавляется кнопка «Фильтры». По нажатию она открывает
компактную выпадающую плашку с категориями-мультивыбором. Кнопка показывает
число выбранных категорий, когда оно больше нуля. По умолчанию не выбрано
ничего, поэтому показаны все рестораны выбранного города с координатами.

При выбранных категориях показываются рестораны, у которых есть хотя бы один
выбранный тег. Отбор применяется одновременно к маркерам, нумерованному
списку, ссылке для маршрута Google Maps и карточкам города. Смена города или
категории снимает фокус с маркера.

## Проверка

TypeScript и production-сборка Vite должны успешно завершаться. Ручная проверка:
выбрать несколько категорий в редакторе, сохранить ресторан, затем убедиться,
что на карте корректно отображаются все совпадающие теги и что основной каталог
не отфильтрован.
