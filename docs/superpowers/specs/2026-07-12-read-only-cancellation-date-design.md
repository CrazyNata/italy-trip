# Неизменяемая дата отмены

## Цель

Убрать со вкладки «Отмена» элемент календаря, который создаёт впечатление, что
дату бесплатной отмены можно изменить прямо в списке.

## Изменения

- Удалить поле `input type="date"` из карточек отмены вместе с обработчиком
  изменения `freeCancel`.
- Оставить справа существующие текстовые дату «бесплатно до» и статус.
- Заменить верхнюю подсказку на «Сроки бесплатной отмены по каждому жилью.».
- Сохранить переход из карточки во вкладку «Жильё» и сортировку по дате.

## Ограничения и проверка

Изменение касается только представления вкладки «Отмена». Сохранённые даты и
возможности редактирования жилья в других местах не меняются. Проверка включает
production-сборку и подтверждение отсутствия календарных полей в карточках.
