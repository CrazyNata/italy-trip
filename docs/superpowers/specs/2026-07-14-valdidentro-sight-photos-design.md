# Фотографии достопримечательностей Вальдидентро

## Цель

Добавить по одной фотографии к семи новым точкам маршрута Валь-Виола и озёр Канкано.

## Подход

- Использовать публичные фотографии Wikimedia Commons с открытыми лицензиями.
- Сохранить URL в поле `photo` существующих записей `Sight`.
- Не менять интерфейс, не загружать файлы в Supabase Storage и не заменять фото других мест.
- Проверить, что ссылка каждого изображения доступна и карточки получают непустое поле `photo`.
