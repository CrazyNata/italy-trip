# Фотографии остановок Стельвио

## Цель

Добавить по одной фотографии Wikimedia Commons ко всем восьми остановкам автомобильного маршрута Стельвио.

## Подход

- Сохранить публичный URL изображения в поле `photo` каждой новой записи `Sight`.
- Использовать изображения соответствующего места или окружающего альпийского ландшафта.
- Не изменять существующие точки, описания, координаты или интерфейс.
