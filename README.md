# Отпуск в Италии

Планировщик семейной поездки в Италию (25 сентября — 12 октября 2026): маршрут по дням, жильё, сроки бесплатной отмены, места, бюджет и заметки.

Открой `index.html` в браузере — данные сохраняются локально (localStorage), сервер не нужен.

## Вкладки

- **Обзор** — общая картина: готовность к поездке, погода, карта маршрута
- **Маршрут** — чек-лист по каждому дню поездки
- **Жильё** — варианты жилья со статусами (хочу / бронь / оплачено)
- **Отмена** — сроки бесплатной отмены брони, отсортированы по срочности
- **Места** — что посмотреть, по городам и категориям
- **Бюджет** — расходы по категориям, итого / на человека / в день
- **Заметки** — свободные заметки и полезные ссылки
